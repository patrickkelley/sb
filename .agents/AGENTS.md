# System Instructions for Imitation of Christ Project

This project focuses on processing, cleaning, and formatting the digital edition of the book **"Imitation of Christ"** by Thomas à Kempis.

## Project Structure
- [raw/imitation_of_christ/imitation_of_christ.md](file:///workspaces/sb/raw/imitation_of_christ/imitation_of_christ.md): The raw scanned text, susceptible to OCR errors, stray marks, page headers, footers, and page numbers.
- [wiki/](file:///workspaces/sb/wiki/): Destination directory for structured, clean wiki pages per book/chapter.
- [output/](file:///workspaces/sb/output/): Destination directory for compiled and formatted outputs.

## Rules & Formatting Guidelines
1. **OCR Cleanup**: Actively identify and fix common OCR errors, stray characters, and incorrect words. Remove page numbers, running headers, and Google watermarks/guidelines from the text during extraction.
2. **Markdown Structure**: Use clear, semantic Markdown headers. Ensure each book and chapter is formatted cleanly with consistent heading levels.
3. **No Trailing Whitespace**: Ensure all files are saved without trailing whitespaces and have a single newline at the end.
4. **Preserve Content**: Do not alter the actual meaning, reflections, or prayers of the text, but ensure formatting is elegant and readable.
